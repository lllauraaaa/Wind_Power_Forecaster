# 风电功率预测（wind_power_Forecaster）

# 面向日前电力交易的概率型风电预测系统 (Probabilistic Wind Power Forecaster)

## 项目概述

本项目旨在为**日前电力市场（Day-Ahead Market）**开发一套高鲁棒性的**风电功率概率预测系统**。

在真实的电力现货交易中，传统的点预测（优化 RMSE/MAE）存在致命缺陷，因为它无法量化气象驱动发电的不确定性。本项目采用**分位数回归（Quantile Regression）**输出完整的概率场景（如 P10, P50, P90）。通过精准预测“欠发”与“超发”的非对称风险，该模型能够直接赋能自动化交易算法，动态制定报价策略（Bidding Strategy），从而最小化电网的**不平衡考核偏差惩罚（Imbalance Penalties）**。

模型管线基于权威的 **GEFCom2014 风电赛道**数据集，完全依赖欧洲中期天气预报中心（ECMWF）提供的纯外生气象预报（NWP）进行驱动。

## 特征工程：纯数据驱动的统计学视角

- **动力学多项式展开 (`WS_100_cubed`)**：构造了风速的三次方特征 ($v^3$)。通过皮尔逊相关系数和互信息严格验证了其在捕获风能方程时的统计学显著性。
- **周期性三角函数编码**：将风向（0°-360°）和时间戳（小时、月份）通过正弦 (`sin`) 和余弦 (`cos`) 映射到连续的空间坐标系。这一优雅的技巧彻底消除了午夜（23点到0点）的数值断层，并自然捕获了**热力学日夜交替规律**（如白天热力对流 vs 夜间低空急流）。
- **垂直风剖面**：融合了 10 米与 100 米双高度气象数据，使模型能够隐式推导大气稳定度与风切变指数。

## 算法架构与递进策略

项目采用企业级模块化架构（`src/data_pipeline.py` -> `src/models/` -> `src/evaluate.py`）。我设计了递进式的建模策略，以论证引入非线性复杂模型的必要性：

1. **统计学基准 (`statsmodels`)**：**线性分位数回归**。确立了基准的 Pinball Loss，并证明了线性假设无法拟合风功率曲线的截断特征（如额定风速后的功率平移）。
2. **核心机器学习 (`LightGBM`)**：**基于分位数目标函数的梯度提升树**。直接针对非对称损失函数进行优化，完美捕获了非线性的“硬阈值”。
3. **深度学习 (`PyTorch`)**：**结合自定义张量损失的 LSTM**。设计了滑动窗口序列模型，并在 PyTorch 底层手写了支持反向传播的 Pinball Tensor Loss。

## 性能评估 (Out-of-Time 严格时序测试)

模型在一个完全隔离的**未来 30 天连续时间段**上进行了预测。

**评估指标: Pinball Loss (分位数损失)**
*(分数越低越好。目标分位数: 10%, 50%, 90%)*


| 模型                                        | 架构本质           | 平均 Pinball Loss | 业务洞察                                                          |
| :------------------------------------------ | :----------------- | :---------------- | :---------------------------------------------------------------- |
| **气候态基准模型（Climatology Benchmark）** | 无条件历史经验分布 | 0.0614            | 不依赖任何特征的历史概率分布。                                    |
| **线性基准 (Linear Quantile Reg)**          | 线性 / 参数化      | 0.0391            | 较气候态基准模型提升 36%。                                        |
| **PyTorch LSTM**                            | 深度序列模型       | 0.0375            | 优于线性基准，但在缺乏自回归特征的纯气象表格数据上受限。          |
| **LightGBM**                                | 非线性树模型集成   | **0.0344** 🥇     | **最佳选择。较**线性基准基准提升 12%。** 极度擅长处理硬阈值交叉。 |

*(**SOTA 水准备注**: GEFCom2014 全球风电赛道冠军队伍的最终得分为 **0.038**。)*

### 为什么在这个场景下 LightGBM 击败了 LSTM？

尽管深度学习能力强大，但 LightGBM 在这个特定的表格式（Tabular）任务中胜出。
在日前预测中，由于缺乏高频实时的自回归输入（$T-1$ 功率），LSTM 的时间序列记忆优势大打折扣，预测过程几乎完全由目标时刻的外生气象状态决定。此外，风功率曲线存在严重的非线性截断（如 25m/s 强制停机掉0）。擅长平滑函数逼近的神经网络在拟合这种“阶梯式突变”时极易挣扎或过拟合；相反，决策树底层的 `IF-ELSE` 空间划分机制天生就完美契合这种“硬阈值”业务规则。

### 交易策略应用 (为什么要预测 P10 和 P90？)

- **高惩罚场景**：如果电网对“欠发（未达承诺电量）”施加严厉的不平衡惩罚，交易算法必须采用保守策略，参考 **P10 预测值**（即有 90% 把握能发出来的保底电量）进行申报。
- 模型输出的分位数概率分布，将直接作为随机规划（Stochastic Programming）求解器的输入，以最大化期望财务收益。
